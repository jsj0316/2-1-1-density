# 🔬 물질의 특성: 밀도 가상실험실
**송쌤과학 × 승재의 과학체육관 공동 제작**

이 프로젝트는 중학교 과학 ‘물질의 특성’ 단원에서 중요한 개념인 **밀도(density)**를  
학생들이 직접 탐구할 수 있도록 만든 **인터랙티브 가상 실험실(Virtual Lab)**입니다.  
여러 물질의 질량과 부피를 측정하고, 그 데이터를 활용해 밀도를 계산하며  
물질을 비교·분류하는 과정을 시각적으로 체험할 수 있습니다.

---

## 🎯 학습 목표
- 밀도의 정의(밀도 = 질량 / 부피)를 이해한다.  
- 질량과 부피를 측정하는 기본 실험 절차를 익힌다.  
- 같은 부피여도 물질에 따라 질량이 달라짐을 탐구한다.  
- 계산한 밀도를 바탕으로 물질을 **식별**하거나 **분류**한다.  
- 물질이 가라앉는지 뜨는지를 밀도 관점에서 설명할 수 있다.

---

## 🧪 주요 기능

### ⚖️ 1. 질량 측정(저울) 시뮬레이션  
- 다양한 물질 샘플을 저울에 올려 실제처럼 질량을 확인할 수 있습니다.  
- 오차 범위를 고려한 사실적인 모델링 제공.

### 🧱 2. 부피 측정(메스실린더) 시뮬레이션  
- 물체를 물 속에 넣었을 때 변화량을 읽어 부피를 측정합니다.  
- 부피 변화를 애니메이션으로 표현해 직관적 이해를 돕습니다.

### 📈 3. 밀도 자동 계산 및 그래프 시각화  
- 측정값을 기반으로 밀도 계산  
- 여러 물질을 비교하는 **막대그래프** 자동 생성  
- 밀도 순서 정렬 기능

### 🧊 4. 물질 식별 모드  
- 밀도표를 참고해 ‘정체를 모르는 물질 샘플’의 정체를 추리하는 활동 제공

### 🏊 5. 부력 시뮬레이션(뜨는 물체 vs 가라앉는 물체)  
- 밀도 차이를 이용한 정성적 이해  
- 실제 물탱크에서 떠오르거나 가라앉는 애니메이션 제공

---

## 🛠️ 기술 스택
- **Frontend:** React, Svelte 중 선택
- **Visualization:** D3.js, Canvas, Three.js  
- **Simulation Engine:** JavaScript 기반 밀도/부력 모델  
- **Backend(선택):** Node.js (활동 데이터 저장)

---

## 📘 교육과정 연계
- 중학교 과학: **물질의 특성 / 질량·부피·밀도 / 물질의 분류**  
- 탐구 과정 중심 수업에 최적화  
- 정량적 측정 → 계산 → 분석 → 설명의 과학적 사고 흐름 제공
